*홍콩주식과 중국주식은 홍콩재무제표, 중국재무제표를 나타냅니다. <br>
*중국주식 이용방법 : cn_fs('코드번호') ex) cn_fs('000002') <br>
*재무재표는 한국재무재표를 나타냅니다. 이용방법 fs('기업명'), ex) kr_fs('삼성전자')
